# Flappy Bird

A feature-rich, physics-defying Flappy Bird game built entirely in modern C++17 and SFML 3! 
Designed by Jinendra Banthia.

## Features

- **Advanced Pipe Mechanics**: The game gets progressively harder! The longer you survive, the more dangerous the obstacles become. You'll encounter normal static pipes, moving pipes, diagonal pipes, crushing pipes, and breakable wooden pipes!
- **Anti-Gravity Mode**: Click the "Gravity: Anti" toggle on the main menu to flip physics completely upside-down!
- **Interactive Start Menu**: Use an intuitive drop-down UI to skip straight to the harder levels right from the start screen.
- **Dash Ability**: Press SHIFT to perform a rapid horizontal dash! This grants temporary invincibility and allows you to shatter breakable wooden pipes!
- **Procedural Pixel-Art Aesthetics**: Features a custom window icon, floating start-screen logo (`logo.png`), a scrolling forest floor, and custom chunky pixel-art clouds dynamically generated by the game engine.
- **Visual Haptics**: The bird sprite features a dynamic squash-and-stretch visual scale pop when jumping, simulating tactile feedback.
- **Procedural Audio**: No external audio files required! The flap and crash sound effects are generated mathematically at runtime.
- **Persistent High Scores**: Your best score is saved directly to `highscore.txt` so it persists between gaming sessions.

## Requirements

- CMake (3.10+)
- MinGW-w64 (or compatible C++17 compiler)
- SFML 3.0.2

## Build Instructions

1. Open a terminal in the root directory.
2. Generate build files:
   ```bash
   cmake -B build
   ```
3. Compile the game:
   ```bash
   cmake --build build
   ```
4. Run the executable found in the `build` directory!

## Controls

### Main Menu
- **Left Click**: Interact with the Start Level dropdown and the Gravity toggle button.
- **SPACE**: Start the game!

### In-Game
- **SPACE**: Flap! (Hold the spacebar to temporarily weaken gravity for more precise long jumps!)
- **SHIFT**: Dash! (Cooldown applies. Use this to blast through breakable wooden pipes.)
- **F**: Toggle Anti-Gravity mode mid-flight!
- **Numbers 1-9**: Developer debug tool to instantly jump to specific levels (and adjust score).
